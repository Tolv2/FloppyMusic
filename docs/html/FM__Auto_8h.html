◆ FM_Auto_StartPlayingSound()

int FM_Auto_StartPlayingSound ( int periodNanoseconds)

Start playing a sound using Automatic managment mode.

Automatic managment simply plays the note on the first floppy that isn't currently playing another note. This works best with a live instrument, as they usually only send over one channel.

Parameters
periodNanosecondsthe period of the sound to play. Note that you must pass the exact same value back to FM_Auto_StopPlayingSound to stop playing the sound started by this call. Also note that a map of MIDI notes to periods is available in the library, and can be safely used here.
