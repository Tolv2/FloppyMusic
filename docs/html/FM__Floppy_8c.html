◆ FM_FloppyLoop()

void * FM_FloppyLoop ( void * _floppy)

Used internally.

Main loop of the floppies thread. One instance of this function is spawned for every registered floppy. It observers the sleepPeriod member of the FM_Floppy struct and handles them accordingy.

Parameters
_floppypointer to FM_Floppy struct that the function belogns to. Declared void* because pthread needs it. Safe to cast.

◆ FM_RegisterFloppy()

int FM_RegisterFloppy ( FM_Floppy * floppy)

Used internally, but can be used externally.

Registers the floppy for the library to use as a playing device. Clears the pins, fills out the rest of the struct, and spawns and starts its thread.

Parameters
floppythe FM_Floppy object which is to be registered.
